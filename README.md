# BLP-cladisticPPMDP
Program
The simulated data for program1 and program2 are contained in files Data_program1 and Data_program2, and the corresponding real data are contained in files Real_666 and Real_1136.

Program1: The F tensor and state trees are the input to this program, and the output is the ancestral graph. In this program, the components of the tensor F^+ are calculated using the tensor F and state trees, and then converted to the matrix F^+, and using the ancestral condition, the ancestral graph is obtained.

Program2: The tensor F and the ancestral graph are the input to this program, and the output is the multi-state phylogenetic tree.
